# Директория с исходными данными

## Структура данных

Файлы в этой директории должны содержать данные об успеваемости студентов в одном из поддерживаемых форматов:

### Поддерживаемые форматы:
- CSV (рекомендуется)
- Excel (.xlsx, .xls)
- JSON

### Обязательные колонки:
1. `student_id` - уникальный идентификатор студента
2. `grade` - оценка (число от 1 до 10)
3. `subject` - название предмета

### Опциональные колонки:
- `date` - дата выставления оценки
- `group` - учебная группа
- `attendance` - посещаемость (0.0-1.0)
- `week` - неделя семестра
- `teacher` - преподаватель
